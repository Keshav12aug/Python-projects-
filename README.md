# Real Estate Data Analysis Project

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a real estate dataset to uncover meaningful insights related to property pricing, locality trends, builder influence, property types, and pricing patterns.

Using Python libraries such as Pandas, Matplotlib, and Seaborn, the project answers multiple business-oriented research questions related to the housing market.

---

## Research Questions and Findings

| Question | Finding |
|---|---|
| Which is the costliest flat in the dataset? | A 6 BHK flat in Sector 42 priced at ₹122.63 Crores in DLF Camellias society |
| Which locality has the highest average price? | Balliawas |
| Which locality has the highest rate per square foot? | Sector 113 |
| Ready-to-move vs Under-construction pricing? | Under-construction properties are costlier on average |
| Does RERA approval affect pricing? | RERA-approved properties command a premium |
| Which BHK configuration is most expensive? | 114 BHK configuration |
| Which property type is the costliest? | Villa |
| Which builders price higher? | Camelliaass, Cameliaas, Tulip, Magnoliaass, Mah |
